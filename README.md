# vue-blog-client

## 🐈 介绍

> 基于vue的多人共享blog

包含首页、用户文章列表、登录、注册、个人管理、编辑、发布等功能。

## 🚀 开始


1.  第一步：git clone 或 download 该项目

2.  第二步：cd 进入项目目录

> 若环境中不存在 `yarn`/`npm`，则需要先[安装yarn](https://yarnpkg.com/zh-Hans/docs/install)

```bash
# 安装依赖包
yarn # 或 npm i

# 开始
yarn dev # 或 npm run dev
```

`yarn dev`运行完后，使用浏览器打开提供的网址
